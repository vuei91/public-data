# 아파트 실거래가 조회 앱

국토교통부 아파트 매매 실거래가 API를 활용한 React 앱입니다.

## 기능

- 지역코드와 거래년월로 아파트 실거래가 리스트 조회
- 아파트 클릭 시 상세 정보 모달 표시
- 거래금액, 위치, 면적, 층수 등 상세 정보 제공

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 환경 변수 설정

`.env` 파일에 API 키를 설정하세요:

```
VITE_SERVICE_API_KEY=your_api_key_here
```
